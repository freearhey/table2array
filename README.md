# table2array

Node.js utility to convert HTML table to Javascript array.

## Installation

```sh
npm install table2array
```

## Usage

```js
import table2array from 'table2array'

const table = `<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Sue</td>
      <td>15</td>
    </tr>
    <tr>
      <td>Steve</td>
      <td>12</td>
    </tr>
  </tbody>
</table>`

const array = table2array(playlist)

console.log(array)
```

Output:

```json
[
  ["Name", "Age"],
  ["Sue", "15"],
  ["Steve", "12"]
]
```

## Examples

### Table with rowspan

<table border="1">
  <thead>
    <tr>
      <th>Parent</th>
      <th>Child</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td rowspan="3">Marry</td>
      <td>Sue</td>
    </tr>
    <tr>
      <td>Steve</td>
    </tr>
    <tr>
      <td>Tom</td>
    </tr>
  </tbody>
</table>

```json
[
  ["Parent", "Child"],
  ["Marry", "Sue"],
  ["Marry", "Steve"],
  ["Marry", "Tom"]
]
```

### Table with colspan

<table>
  <thead>
    <tr>
      <th>Monday</th>
      <th>Tuesday</th>
      <th>Wednesday</th>
      <th>Thursday</th>
      <th>Friday</th>
      <th>Saturday</th>
      <th>Sunday</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td colspan="5">8:00 - 17:00</td>
      <td>10:00 - 13:00</td>
      <td>Closed</td>
    </tr>
  </tbody>
</table>

```json
[
  ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"],
  [
    "8:00 - 17:00",
    "8:00 - 17:00",
    "8:00 - 17:00",
    "8:00 - 17:00",
    "8:00 - 17:00",
    "10:00 - 13:00",
    "Closed"
  ]
]
```
